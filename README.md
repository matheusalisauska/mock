## MOOCK API

A full-stack project that provides API endpoints with mocked and randomly generated data for integration testing, built with **Next.js, oRPC, Prisma, React Query and Better Auth.**

## Features

- Project-based organization with custom entities and dynamic fields
- Mocked and random data generation using Faker.js
- Configurable API response delays and error scenarios
- API endpoints designed for integration and E2E testing

## Platform & Infrastructure Features

- Authentication and authorization system
- Subscription-based plans with usage limits
- Access control based on user subscription plan
- Billing logic and payment provider integration

## Architecture Notes

- **Feature-based architecture** for better scalability and maintainability
- End-to-end type safety using **oRPC**
- Centralized data fetching and error handling via React Query
